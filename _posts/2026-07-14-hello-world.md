## How this works

Every post is just a text file. To publish a new one, I create a file in the `_posts` folder named like `2026-07-20-my-post-title.md`, starting with these three lines:

```
---
layout: post
title: "My Post Title"
---
```

Then I write in plain Markdown below that:

- **Bold text** with double asterisks
- *Italics* with single asterisks
- [Links](https://www.markdownguide.org/basic-syntax/) with brackets and parentheses

> Blockquotes look like this.

That's the whole workflow. Upload the file, and GitHub publishes it in about a minute.
